# eslint-config-wdfe

[![npm package](https://img.shields.io/npm/v/eslint-config-wdfe.svg)](https://www.npmjs.org/package/eslint-config-wdfe)

Eslint config for wdfe

## Install

```bash
npm i eslint-config-wdfe --save-dev
```

.eslintrc.json
```json
{
    "extends": "wdfe",
    "parser": "babel-eslint"
}
```
